# General

- No Javascript is allowed. Only CSS animations and styles can be used

# Color Palette & Structure

- **Backgrounds:** Use `#ffffff` for cards/content panels, and a very soft gray like `#f8f9fa` or `#f1f3f4` for the main canvas background.
- **Borders:** Google uses flat borders instead of shadows. Use `.border`, `.border-light`, and completely strip out shadows using `.shadow-none`.
- **Typography:** Google uses Roboto or Open Sans. Import Roboto via Google Fonts and set it as the base stack.

# Key UI Tricks to Keep It "GCP-Clean"

- **Font Sizing:** Keep text small (13px to 14px for body/tables) and lean into `text-muted` or gray shades (`#5f6368`) for subtext.
- **Icons:** Use Google Material Symbols (or Material Icons) exclusively for the sidebar and actions.
- **Primary Accent:** Use the signature Google Cloud blue (`#1a73e8`) for primary buttons, active links, and selections.
